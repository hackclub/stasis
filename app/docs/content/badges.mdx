## Badges

Badges are achievements you earn by incorporating specific hardware features into your projects. They showcase your skills and add variety to your builds.

### Available Badges

| Badge | Description |
|-------|-------------|
| I2C | Used the I2C communication protocol |
| SPI | Used the SPI communication protocol |
| WiFi | Incorporated wireless networking |
| Bluetooth | Used Bluetooth connectivity |
| USB | Implemented USB communication |
| CAD | Created custom CAD designs |
| PCB | Designed a custom printed circuit board |
| 3D Print | Used 3D printed parts |
| Laser Cut | Used laser-cut components |
| Motor | Incorporated motors or servos |
| Sensor | Used sensors for input |
| Display | Added a screen or display |
| Audio | Implemented audio input/output |
| Battery | Designed battery-powered operation |
| Solar | Used solar power |
| LED | Incorporated LED lighting |

### Claiming Badges

When creating or editing your project, select the badges that apply. During build review, reviewers verify that your project actually uses these features based on your journal entries and documentation.

> Only claim badges for features you've genuinely implemented. Reviewers will check your work.
